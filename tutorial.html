① 记一条 —— 一句话录入Capture — one sentence

iOS:底部「记一条」tab(或首页右上角 ✏️)。macOS:主窗口顶部输入框(⌘N 聚焦)。

iOS: the Record tab (or ✏️ top-right on Home). macOS: the input bar (⌘N).

直接用自然语言写,app 自动拆出名字、层级位置、型号、颜色、容量、购买信息:

Write naturally; the app splits out name, nested location, model, color, capacity, purchase info:

  • 充电宝在卧室五斗柜第二格抽屉
  • 护照:书房保险箱
  • Sony WH-1000XM5 黑色 去年在京东买的 放客厅电视柜
  • passport in the study safe · 3 chargers on the TV stand
  • 一次记多条:换行、逗号、分号、「然后 / 还有」都能分隔。
  • Batch capture: separate with newlines, commas, or semicolons.
  • 语音输入(iOS):点「记下来」旁的 🎤,说话实时转文字。
  • Voice input (iOS): tap the 🎤 next to Save and speak.
  • 位置可以省略:先记下,详情页里随时补。
  • Location is optional — add it later from the detail page.
  • 同名智能处理:再次输入已有物品会弹窗问你是「更新位置 / 补充信息」还是「另起一条」;写「X 的型号是 Y」直接更新已有物品字段。
  • Duplicates handled: re-entering an existing item asks whether to update it or create anew; "X's model is Y" updates fields in place.

③ 详情页:位置、时间线、借出Detail: location, timeline, lending

「用过吗?」四个位置按钮 —— 点开物品通常意味着你在找它,用完顺手告诉 app:

Four location buttons — opening an item usually means you're looking for it; tell the app what happened:

  • 🟢 还在原位 · 🔵 放回原位了 · 🟠 位置变了…(改新位置) · 🔴 不知道在哪
  • 🟢 Still there · 🔵 Put it back · 🟠 Moved (enter new spot) · 🔴 Lost track
  • 每次动作都进时间线:位置变化 + 字段修改(含 AI 修改,可一键还原名字)全记录。
  • Every action lands on the timeline: moves + field edits (incl. AI edits, with one-tap name restore).
  • 借出:紫色「借给…」按钮记下借给谁;归还点顶部橙色徽章上的「归还」。借出的东西列表里有紫色标记,筛选面板有「借出」一栏,搜索借的人名字也能找到。
  • Lending: the purple "Lend to…" button records the borrower; return via the orange badge. Lent items are marked, filterable, and searchable by borrower name.
  • 关联物品:把配件和主机连起来(如 iPad 和它的笔),详情页互相跳转,一组最多 8 件。
  • Related items: link accessories to the main device; jump between them; up to 8 per group.
  • 照片:编辑页添加,详情页点击全屏、双指缩放。
  • Photos: add in the editor; tap for full screen, pinch to zoom.

④ 搜索与筛选Search & filter

  • 搜索覆盖:名字、备注、位置路径、型号、颜色、渠道、容量、借出人、标签名。iOS 下拉出搜索框;macOS ⌘F。
  • Search covers name, notes, location path, model, color, source, capacity, borrower, and tag names. Pull down on iOS; ⌘F on macOS.
  • 统计瓷砖即筛选器(iOS):点「房间/渠道/品牌/年份」瓷砖下拉选一个,点「置顶/借出」直接筛,再点一次取消,点「物品」清除全部;facet 行(macOS:房间、位置、借出、渠道、年份、品牌)同理可叠加,比如「卧室 + 借出去的」。
  • Stat tiles are filters (iOS): tap Room/Source/Brand/Year tiles for a dropdown, tap Pinned/Lent to toggle, tap again to clear, tap Items to clear all; facet rows (macOS) combine the same way — e.g. "bedroom + lent out".
  • 位置是树状的:房间 > 家具 > 格子。同名位置自动复用,改名整棵子树跟着变;macOS 设置 → 位置里可整理(合并/重命名/删除)。
  • Locations form a tree: room > furniture > drawer. Same-name spots are reused; renames cascade; tidy them up in macOS Settings → Locations.

⑤ iCloud 同步(多设备)iCloud Sync

Mac 和 iPhone 登录同一个 Apple 账户即可,零配置。

Sign in with the same Apple Account on both devices — zero config.

  • 实时同步:开关默认打开(设置 → iCloud 同步),改动自动在设备间流动,通常几秒到一分钟。
  • Live sync is on by default (Settings → iCloud Sync); changes flow across devices within seconds to a minute.
  • 手动同步:iOS 首页下拉刷新或点右上角 ☁️;设置 → 数据里也有「立即同步」。它会拉取其它设备写到 iCloud 云盘的数据文件、合并去重(同名同位置只留一条),再把本机最新数据写回去。
  • Manual sync: pull-to-refresh or tap ☁️ on iOS Home (also "Sync Now" in Settings → Data). It pulls the data file other devices wrote to iCloud Drive, merges with de-duplication, and writes back.
  • 看得见的备份:数据文件就在 iCloud 云盘/Whereabouts/ 里(「文件」app / 访达可见),退出 app 时自动更新。
  • A visible backup: the file lives in iCloud Drive/Whereabouts/ (visible in Files / Finder), refreshed on quit.
🔒 全部走你自己 Apple 账户的私有空间,开发者永远看不到任何内容。不登录 iCloud 也能用 —— app 自动落回纯本机模式。
🔒 Everything stays in your own Apple Account's private space; the developer can never see it. No iCloud? The app just runs in local-only mode.

⑥ AI 理解(可选,但强烈推荐)AI understanding (optional, recommended)

  • 写得再随意也能拆得干干净净,还会自动挑分类标签、修错别字、把「门口电梯间鞋架上方」拆成正确的位置层级。
  • Messy input parses cleanly; it also picks a category tag, fixes typos, and splits compound locations correctly.
  • 配置只要 10 分钟:图文配置教程(火山引擎 / Claude 两条路线)。测试连接成功后,录入自动 AI 理解会自动打开。
  • Setup takes ~10 minutes: see the illustrated setup guide. Once the connection test passes, AI-on-input turns on automatically.
  • 费用:理解一件约 2 分钱;火山引擎新用户送大量免费额度,轻度使用这辈子花不完。用量和估算费用在「用AI理解」设置页里随时可查。
  • Cost: ~half a cent per item; Volcengine's free tier lasts ages. Usage & cost estimates live in the AI settings page.
  • 手动触发:列表长按(iOS)/ 右键(macOS)→「用 AI 理解」,老数据也能批量洗一遍。理解完成后行尾出现 ✓ 和「一键还原」;时间线里 AI 改过的名字也随时可还原。
  • Manual trigger: long-press (iOS) / right-click (macOS) → "Understand with AI". After it finishes, a ✓ and an Undo pill appear on the row; AI renames can always be reverted from the timeline.

⑦ 标签 · 通知 · 数据Tags · Reminders · Data

标签

Tags

14 个预设分类开箱即用;「设置 → 标签管理」里可改名、换色(9 色调色板)、删除、新建。录入时按物品名自动建议;AI 开启后由 AI 权威分类。

14 presets out of the box; rename/recolor/delete/create under Settings → Manage Tags. Auto-suggested on input; AI takes over classification when enabled.

定期提醒

Reminders

置顶的重要物品可收「还在原位吗?」本地通知:频率(每天/每周/每月)、时间、文案模板都可自定义;点通知直达该物品。完全本地,无网络。

Pinned items can get "still there?" local notifications — frequency, time, and message template are customizable; tapping jumps to the item. Fully local.

数据进出

Data in & out

设置 → 数据:全量 JSON 导出(含照片)/ 导入(默认去重)。数据永远是你的,随时带走。

Settings → Data: full JSON export (photos included) / import (dedup by default). Your data is always yours.
