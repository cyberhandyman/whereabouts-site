AI Setup — the Hand-holding Guide

「何处」不配 AI 也完全能用。但配上一把 AI 钥匙后,随手写的一句「充电宝在卧室抽屉第二格」会被拆得更聪明:型号、颜色、位置层级、分类标签全都自动就位。这页教你从零开始拿到这把钥匙 —— 全程大约 10 分钟,跟着点就行。

Whereabouts works fine without AI. With an AI key, though, a quick note like "power bank in the 2nd bedroom drawer" gets parsed much more intelligently — model, color, nested location, and category tag all filled in automatically. This page walks you from zero to a working key in about 10 minutes.

路线 A · 火山引擎,五步拿到钥匙Route A · Volcengine in five steps推荐

  1. 注册火山引擎账号Create a Volcengine account

    打开 volcengine.com,右上角「注册」,用手机号收验证码即可。它是字节跳动(抖音母公司)的云服务,豆包大模型就在这里。

    Open volcengine.com and sign up with a phone number. It's ByteDance's cloud platform, home of the Doubao models.

  2. 完成实名认证Complete real-name verification

    调用大模型 API 之前必须实名。控制台会引导你走「个人实名认证」:填身份证号 + 支付宝或微信扫码刷脸,一般几分钟内通过。

    Real-name verification is mandatory before calling the API. The console guides you through it — ID number plus an Alipay/WeChat face scan; usually approved within minutes.

  3. 打开「火山方舟」控制台,创建 API KeyOpen the Ark console and create an API Key

    直达链接:console.volcengine.com/ark。在左侧菜单找到「API Key 管理」→ 点「创建 API Key」。

    Go to console.volcengine.com/ark → left sidebar → API Key management → Create API Key.

    ⚠️ 完整密钥只在创建时显示一次 —— 立刻复制,粘到备忘录暂存。丢了也不要慌,删掉重建一把即可。
    ⚠️ The full key is shown only once at creation — copy it immediately. If you lose it, just delete and create a new one.
  4. 确认模型可用(新用户有免费额度)Make sure the model is enabled (free quota for new users)

    新用户每个模型送 50 万 tokens 免费推理额度,足够「何处」用几百次。免费额度用完后,去「开通管理」页把要用的模型点开通,再到费用中心用支付宝/微信充值(几块钱就够用很久,见下方费用表)。

    New users get 500K free tokens per model — good for hundreds of Whereabouts calls. When it runs out, enable the model on the 开通管理 (Model activation) page and top up via Alipay/WeChat.

  5. 把三样东西填进「何处」Enter three things into Whereabouts

    看下面「填进 app」小节 —— 服务商选「火山引擎」,Model 填 doubao-seed-1-6-250615(或控制台里你看到的其它在售模型 ID / ep- 开头的接入点 ID),API Key 粘贴刚才复制的,Endpoint 保持默认不动。

    See "Enter into the app" below — pick Volcengine as provider, set Model to doubao-seed-1-6-250615 (or any live model ID / an ep- endpoint ID from your console), paste the API Key, leave Endpoint at its default.

路线 B · Claude 官方,四步拿到钥匙Route B · Claude API in four steps

  1. 注册 Claude 开发者控制台Sign up for the Claude Console

    打开 platform.claude.com(旧地址 console.anthropic.com 会自动跳转过来),用邮箱或 Google 账号注册并验证邮箱。注意:这个「开发者控制台」和 Claude 聊天网页是两套账户体系,Claude Pro 会员≠API 额度。

    Open platform.claude.com (the old console.anthropic.com redirects here). Sign up with email or Google. Note: the developer console is separate from the Claude chat subscription — Claude Pro does not include API credits.

    ⚠️ Anthropic 官方支持地区不含中国大陆和香港。你需要处于支持地区的网络环境,并用对应地区的支付卡。
    ⚠️ Anthropic's supported-regions list does not include mainland China or Hong Kong. You'll need to be in a supported region (network and payment card).
  2. 充值 Credits(最低 $5)Buy credits ($5 minimum)

    控制台里进 Billing / Plans & billing,绑定 Visa / MasterCard 等国际信用卡,买预付 credits,最低 $5。Credits 一年内有效、不退款 —— 对「何处」来说 $5 够用非常久(见费用表)。没充值前无法创建 API Key。

    In Billing, add a major credit card and buy prepaid credits ($5 minimum; valid for one year, non-refundable). You can't create an API key before purchasing credits. For Whereabouts, $5 lasts a very long time.

  3. 创建 API KeyCreate an API Key

    控制台 → Settings → API keys → Create Key,起个名字(比如 whereabouts),生成一串 sk-ant- 开头的密钥。

    Console → Settings → API keys → Create Key. Name it (e.g. "whereabouts") and you'll get a key starting with sk-ant-.

    ⚠️ 同样只显示一次,立刻复制保存。
    ⚠️ Also shown only once — copy it right away.
  4. 填进「何处」,模型选 Haiku 4.5Enter into Whereabouts; pick Haiku 4.5

    服务商选「Claude」,模型用默认的 Haiku 4.5(最快最便宜,这个场景完全够用),API Key 粘贴,Endpoint 保持默认。

    Pick Claude as provider, keep the default Haiku 4.5 model (fast, cheap, plenty for this job), paste the key, leave Endpoint at its default.

FAQ

我用第三方中转站 / 代理服务,能接吗? Can I use a third-party relay / proxy endpoint?

可以。把设置里的 Endpoint 换成中转站给你的完整 URL 即可(Claude 填 Messages 兼容地址,火山那格填 OpenAI 兼容的 chat/completions 地址)。中转站的质量和隐私请自行甄别 —— 你的物品描述会经过它。

Yes — replace the Endpoint field with your relay's full URL (Anthropic Messages-compatible for Claude; OpenAI chat/completions-compatible for the Volcengine slot). Vet the relay yourself: your item text passes through it.

模型该选哪个? Which model should I pick?

这个任务(拆字段 + 挑标签)不难,便宜的就够:Claude 选 Haiku 4.5;火山选 doubao-seed 系列即可。想要极致准确再上 Sonnet / Opus。

This task (field splitting + tag picking) is easy. Cheap models are plenty: Haiku 4.5 on Claude, or a doubao-seed model on Volcengine. Go Sonnet/Opus only if you want maximum accuracy.

Mac 和 iPhone 要各填一次吗? Do I need to enter the key on both Mac and iPhone?

要。目前设置只存在各自设备本机(没有云同步),所以两台设备各粘贴一次同一把 Key 就好。

Yes. Settings live on each device locally (no cloud sync yet), so paste the same key once per device.

AI 会看到我的哪些数据? What data does the AI see?

只有你主动触发理解的那条物品的文字信息(名字、备注、位置、你写的原句)和你的标签名列表,发给你自己选的服务商。照片不会发送;不触发 AI 就什么都不发。

Only the text of the item you explicitly ask it to understand (name, notes, location, your original sentence) plus your tag names, sent to the provider you chose. Photos are never sent; nothing is sent unless you trigger AI.

不想用了,怎么关掉? How do I turn it off?

把设置里的 API Key 清空即可;录入框下的「用 AI 自动理解」开关也可以单独关。清空后 app 回到纯本地解析,一切照常。

Just clear the API key in Settings (or untick "Understand with AI on input"). The app falls back to fully local parsing.
